resize (rra_num, options)

Writes a new file ‘resize.rrd‘

You will need to know the RRA number, starting from 0: 

rrd.resize(0, :grow => 10.days)
[show source] 
    # File lib/rrd/base.rb, line 69
69:     def resize(rra_num, options)
70:       info = self.info
71:       step = info["step"]
72:       rra_step = info["rra[#{rra_num}].pdp_per_row"]
73:       action = options.keys.first.to_s.upcase
74:       delta = (options.values.first / (step * rra_step)).to_i # Force an integer
